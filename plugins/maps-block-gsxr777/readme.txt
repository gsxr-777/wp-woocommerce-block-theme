== Description ==

Insert a Yandex or Google Map into the Gutenberg editor.

== Installation ==

= Minimum Requirements =

* WordPress 5.0 or greater

= From within WordPress =
1. Visit 'Plugins > Add New'
1. Search for 'Map Block gsxr777'
1. Activate Map Block gsxr777 from your Plugins page.

= Manually =
1. Upload the entire map-block folder to the /wp-content/plugins/ directory.
1. Activate the plugin through the Plugins menu in WordPress.
